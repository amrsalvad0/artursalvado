🚀 COMO INICIAR A APLICAÇÃO:

📁 MÉTODO MAIS FÁCIL:
   → Duplo clique em: "ABRIR-OFFICE-MANAGER.bat"
   → A aplicação abre automaticamente no browser!

⚡ OUTROS MÉTODOS:
   → "iniciar-office-manager.bat" - Script com verificações
   → "iniciar-office-manager.ps1" - Script PowerShell avançado

💻 LINHA DE COMANDO:
   → npm install (primeira vez)
   → npm start

🌐 ACESSO:
   → URL: http://localhost:3000
   → Abre automaticamente no browser
